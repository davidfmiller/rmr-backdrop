Backdrop

Backdrop is a simple YUI module to preload an image and gracefully apply it as the background elment for the body of the page. There are three parameters in the object's constructor configuration, although only the first is required:

url
the path to the image that should be applied as the background image
id (optional)
the id to apply the element ; defaults to "backdrop"
duration (optional)
the length of time (in seconds) taken to introduce the backdrop; defaults to 1 second

One custom event is supported:

drop
fired when the image is loaded and introduction is complete

An example implementation follows:

new Y.Backdrop({
  url : 'http://davidfmiller.github.com/assets/img/backdrop.jpg',

  duration : 1,
}).on('drop', function(e) { Y.log('dropped'); });
